About this page

Use this as a starter UI for the PantryScan API. The API is CORS-enabled, so you can load this file locally and hit the API running via dotnet run.

Actions implemented:

  • List items (GET /items)
  • Add item (POST /items)
  • Basic validation and error display

Tip: if your API runs on HTTPS, set the base URL accordingly (for example, https://localhost:5001).
